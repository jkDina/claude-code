# Token Reminder Plugin

A simple Claude Code plugin that reminds you to check token usage at the start of each session.

## What it does

Displays a reminder at session start about using the `/cost` command to check token usage.

## Installation

1. Copy this plugin folder to your Claude Code plugins directory
2. Enable the plugin in your Claude Code settings

## Hook

- **SessionStart**: Displays a reminder banner about the `/cost` command
